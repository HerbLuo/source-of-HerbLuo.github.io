## Quick Start

### Create a new post

``` bash
$ hexo new "My New Post"
```

### Create a new post

``` bash
$ hexo new "My New Post"
```

### Create a new page

``` bash
$ hexo new page "page"
```

### Run server

``` bash
$ hexo server -l
```

### Generate static files

``` bash
$ hexo generate
```

### Deploy to github

``` bash
$ hexo clean ; hexo generate ; hexo deploy
```
